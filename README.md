# Mind-Land Back-End
This app was build for Mind Land's backend.

## How to Run on Local
Clone this repository (branch - dev)
```
git clone -b dev https://github.com/Mind-Land/mindland-be.git yourfoldername
```
Go to the root of folder
```
cd yourfoldername
```
Install all dependency
```
npm install
```
Run in dev mode
```
npm run dev
```
