# Mind-Land Back-End
This app was build for Mind Land's backend.

## How to run on local
Clone this repository (branch - PunyaYanuar)
```
git clone -b PunyaYanuar https://github.com/OnyanPokon/Mind-Land.git
```
Go to the root of folder
```
cd yourfoldername
```
Install all dependency
```
npm install
```
Run in dev mode
```
npm run dev
```
